# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-22

### Added
- Initial release of n8n-nodes-ms-onedrive-business
- Microsoft OneDrive Business node with file and folder operations
- Microsoft OneDrive Business Trigger node with webhook support
- Support for User Drives and SharePoint Site Drives
- OAuth2 authentication via Azure AD
- File operations: Upload, Download, Get, Delete, Rename, Search, Share
- Folder operations: Create, Delete, Get Items, Rename, Search, Share
- Trigger events: File Created, File Updated, Folder Created, Folder Updated
- Advanced deduplication for trigger node to prevent duplicate executions
- State persistence for reliable change tracking
